This is a test/example file for the module `print_sorted`

to use it, firstly import the function
>>> MyList = __import__('1-my_list').MyList

Then use it in testing for values

>>> my_list = MyList()
>>> my_list.append(1)
>>> my_list.append(4)
>>> my_list.append(2)
>>> my_list.append(3)
>>> my_list.append(5)
>>> print(my_list)
[1, 4, 2, 3, 5]
>>> my_list.print_sorted()
[1, 2, 3, 4, 5]
>>> print(my_list)
[1, 4, 2, 3, 5]

clear my_list
>>> del my_list

Test for negative integers
>>> my_list = MyList()
>>> my_list.append(-1)
>>> my_list.append(1)
>>> my_list.append(-3)
>>> my_list.append(-2)
>>> my_list.append(4)
>>> my_list.append(-7)
>>> my_list
[-1, 1, -3, -2, 4, -7]
>>> my_list.print_sorted()
[-7, -3, -2, -1, 1, 4]
